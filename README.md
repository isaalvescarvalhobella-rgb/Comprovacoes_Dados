# 🚀 Portfólio de Análise de Dados: Performance de Vendas

Bem-vindo ao meu projeto de análise de dados. Neste repositório, explorei um **cenário de negócios simulado** para demonstrar a aplicação prática de quatro frentes tecnológicas integradas.

### 📁 Estrutura do Repositório

* **[📊 01. Análise em Excel](./01_Excel_Vendas/README.md):** Tratamento de dados, Tabelas Dinâmicas e Insights Financeiros.
* **[🗄️ 02. Estruturação SQL](./02_SQL_Vendas/README.md):** Modelagem de banco de dados e consultas estruturadas.
* **[🐍 03. Automação em Python](./03_Python_Vendas/README.md):** Scripts para análise exploratória e automação utilizando Jupyter Notebook.
* **[🖥️ 04. Visualização de Dados](./04_Dashboard_Vendas/README.md):** Dashboard interativo desenvolvido para suporte à decisão.

## 🎯 Objetivo do Projeto
Demonstrar o domínio do ciclo completo do dado através de uma base simulada: desde a limpeza inicial no Excel, migração para bancos SQL, análise estatística com Python e a entrega final de valor através de dashboards estratégicos.

## 🛠️ Competências Aplicadas
* **Data Cleaning & Wrangling** (Excel/Pandas)
* **Manipulação de Bancos Relacionais** (SQLite/SQL)
* **Visualização de Dados & Storytelling** (Looker Studio/Dashboards)
